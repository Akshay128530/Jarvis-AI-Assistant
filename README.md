# 🤖 Jarvis AI Assistant

A voice-controlled desktop assistant built with **Python** that can recognize speech, respond naturally using **Edge TTS**, open websites, and read the latest news using voice commands.

---

## ✨ Features

- 🎤 Wake word detection ("Jarvis")
- 🗣️ Speech Recognition
- 🔊 Natural Voice Responses (Edge TTS)
- 🌐 Open Google
- ▶️ Open YouTube
- 💬 Open WhatsApp Web
- 📸 Open Instagram
- 📘 Open Facebook
- 💼 Open LinkedIn
- 🐦 Open Twitter / X
- 📰 Read Latest News Headlines
- ⚡ Fast voice command processing

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- Edge TTS
- Pygame
- Requests
- NewsAPI
- Webbrowser
- Asyncio

## 🚀 Installation

### Clone the repository

git clone https://github.com/Akshay128530/Jarvis-AI-Assistant.git

### Move into the project

cd Jarvis-AI-Assistant


### Install dependencies

pip install -r requirements.txt

### Run the project

python main.py

## 🎙️ Voice Commands

| Command        | Action 
|----------------|--------------------
| Jarvis         | Activate assistant 
| Open Google    | Opens Google 
| Open YouTube   | Opens YouTube 
| Open WhatsApp  | Opens WhatsApp Web 
| Open Instagram | Opens Instagram 
| Open Facebook  | Opens Facebook 
| Open LinkedIn  | Opens LinkedIn 
| Open Twitter   | Opens Twitter/X 
| Tell me news   | Reads the latest news headlines
